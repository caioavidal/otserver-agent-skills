# OT Servers Agent Skills

A collection of reusable agent skills for Open Tibia servers (otservers), designed to work with any AI coding agent.

## What's inside

Skills are organized by domain. Each skill is a self-contained unit that an agent can use to perform specific tasks.

| Directory | Description |
|-----------|-------------|
| `tibia-house-systems/` | Player-facing Tibia house rules, behavior validation, feature specification, and test scenario guidance. |
| `tibia-conditions/` | Tibia condition, status effect, and icon behavior guidance. |
| *(more coming)* | |

## Usage

Skills are loaded and used by your coding agent. Usage depends on the agent — typically you reference the skill file by path in your prompt or tool configuration.

## Contributing

New skills are welcome! Keep them focused, well-documented, and reusable.
